# PG-forecasting
Econometric analysis and 10-year revenue forecasting for The Procter &amp; Gamble Company (P&amp;G), utilizing time-series modeling (ARIMA/SARIMAX), production function estimation, and structural break analysis to identify long-term growth drivers.
